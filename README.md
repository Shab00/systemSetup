# 🚀 Dev Environment Setup

Quick setup for my terminal environment on any Mac. Gets you tmux, vim, and zsh configured in minutes.

## 📦 What's Included

- **Zsh** - Shell config with powerlevel10k theme
- **Vim** - Editor with plugins (NERDTree, fzf, auto-pairs, tmux navigation)
- **Tmux** - Terminal multiplexer with catppuccin/dracula themes
- **Git** - Global config and ignore rules

## 🏃 Quick Start

```bash
# 1. Clone this repo
git clone https://github.com/Shab00/systemSetup.git
cd ~/systemSetup

# 2. Run setup
chmod +x setup.sh
./setup.sh
